## Environment

- Node.js >= 14.0.0
- Pnpm >= 6.0.0
- React >= 18.2.0
- TypeScript >= 5.3.0
- React Redux >= 9.0.0
- Ahooks >= 3.7.0
- React-Router >= 6.0.0

- Axios >= 1.6.0
